Программа-транслитератор. 
В данной папке хранится:
В папке "CLIENT" хранится клиентское приложение, 
реализованное на языке C#.
В исходном файле index.php находится реализация серверной части
на языке PHP.
В файле "Creation of database" находится SQL-скрипт,
позволяющий создать базу данных для логов программы,
например, с помощью phpMyAdmin.
Для запуска на вашем компьютере, необходимо заменить
некоторые переменные в файла index.php и Input.cs,
изменения указаны в данных файлах.
Также в папке находится три текстовых файла для тестов.
